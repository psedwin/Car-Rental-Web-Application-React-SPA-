# Car Rental SPA Website

## Overview
This project is a responsive single-page car rental website built using HTML, CSS, JavaScript, and React-based content generating functions. The application demonstrates reusable front-end development techniques, interactive user input validation, JSON-driven content, and responsive web design.

## Features
- Responsive single-page application layout
- Fixed navigation bar and footer
- Dynamic navigation routing
- Reusable React components
- JSON-powered vehicle and content lists
- Interactive edit areas with validation
- User input validation for:
  - Strings
  - Numbers
  - Integers
  - Dates
  - Radio buttons
  - Select menus
- Local storage integration
- Responsive mobile-friendly design
- Reusable slideshow/list functionality

## Technologies Used
- HTML5
- CSS3
- JavaScript
- React
- JSON
